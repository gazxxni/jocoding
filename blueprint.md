# AI 동물상 테스트 사이트 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 Google의 Teachable Machine으로 생성된 인공지능 모델을 사용하여 사용자의 얼굴이 강아지상에 가까운지, 고양이상에 가까운지를 판별해주는 '동물상 테스트' 웹 애플리케이션입니다. shadcn/ui의 디자인 철학에 영감을 받아, 미니멀하고 모던한 UI를 통해 사용자 경험을 극대화하며, Google AdSense를 통한 광고 수익화를 목표로 합니다.

## 2. 주요 기능

- **AI 모델 연동:** Teachable Machine으로 학습된 강아지/고양이 이미지 분류 모델을 사용합니다.
- **이미지 파일 업로드:** 사용자가 로컬 장치에서 이미지 파일을 선택하여 업로드할 수 있습니다.
- **모던 UI/UX:** shadcn/ui 스타일에서 영감을 받은 깔끔하고 세련된 디자인을 적용합니다.
- **결과 시각화:** 분석 결과를 텍스트 기반의 깔끔한 형태로 제공하여 가독성을 높입니다.
- **테마 변경:** 라이트 모드와 다크 모드를 모두 지원하며, 각 모드에 최적화된 색상 팔레트를 사용합니다.
- **Google AdSense 통합:** 페이지에 Google AdSense 광고를 추가하여 수익화를 지원합니다.

## 3. 개발 계획 (현재 진행 중인 변경 사항)

1.  **Google AdSense 통합:**
    -   `index.html` 파일의 `<head>` 섹션에 `adsbygoogle.js` 스크립트와 `google-adsense-account` 메타 태그를 추가합니다.
    -   `index.html` 파일의 `card-content` 섹션에 광고 단위(`ins` 태그)를 추가합니다. 이 광고 단위는 이미지 미리보기 및 결과 표시 아래에 위치합니다.
    -   프로젝트 루트 디렉토리에 `ads.txt` 파일을 생성하고 제공된 AdSense 정보를 기록합니다.
    -   `style.css`에 `.ad-unit` 클래스에 대한 스타일을 추가하여 광고 단위가 중앙에 배치되도록 합니다.

2.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
