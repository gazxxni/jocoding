# AI 동물상 테스트 사이트 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 Google의 Teachable Machine으로 생성된 인공지능 모델을 사용하여 사용자의 얼굴이 강아지상에 가까운지, 고양이상에 가까운지를 판별해주는 '동물상 테스트' 웹 애플리케이션입니다. 사용자가 웹캠을 통해 실시간으로 자신의 얼굴을 비추면, AI 모델이 얼굴을 분석하여 각 동물상에 대한 확률을 시각적으로 보여줍니다.

## 2. 주요 기능

- **AI 모델 연동:** Teachable Machine으로 학습된 강아지/고양이 이미지 분류 모델(`https://teachablemachine.withgoogle.com/models/FbPUvPxyt/`)을 사용합니다.
- **실시간 웹캠 분석:** 사용자의 웹캠 화면을 실시간으로 받아와 AI 모델로 분석합니다.
- **결과 시각화:** 분석 결과를 각 항목(강아지, 고양이)에 대한 확률 값과 함께 진행률 표시줄(Progress Bar) 형태로 시각적으로 제공합니다.
- **테마 변경:** 사용자의 편의를 위해 다크 모드와 라이트 모드 테마 전환 기능을 제공합니다.

## 3. 개발 계획 (현재 진행 중인 변경 사항)

1.  **HTML 구조 변경 (`index.html`):**
    -   페이지 제목을 "동물상 테스트"로 변경하고, 그에 맞는 헤더(제목, 설명)를 추가합니다.
    -   기존의 모든 콘텐츠를 제거하고, AI 모델을 시작하기 위한 '시작하기' 버튼, 웹캠 화면이 표시될 `div`, 분석 결과가 표시될 `div`를 배치합니다.
    -   TensorFlow.js와 Teachable Machine Image 라이브러리 스크립트를 `<head>`에 추가합니다.

2.  **CSS 스타일링 (`style.css`):**
    -   기존의 모든 스타일을 제거합니다.
    -   '동물상 테스트'라는 주제에 맞는 재미있고 깔끔한 UI 디자인을 적용합니다.
    -   웹캠 컨테이너, 결과 표시 레이블, 특히 각 동물상 확률을 보여주는 진행률 표시줄에 대한 스타일을 중점적으로 디자인합니다.
    -   다크/라이트 모드 테마를 새로운 UI에 맞게 재구성합니다.

3.  **JavaScript 기능 구현 (`main.js`):**
    -   기존의 모든 JavaScript 코드를 제거합니다.
    -   사용자가 제공한 Teachable Machine 연동 코드를 붙여넣습니다.
    -   `init` 함수: '시작하기' 버튼 클릭 시 웹캠을 설정하고 AI 모델을 로드합니다.
    -   `loop`, `predict` 함수: 웹캠 프레임을 지속적으로 받아와 모델로 예측하고, 결과를 `label-container` 내부의 진행률 표시줄에 업데이트합니다.
    -   다크/라이트 모드 토글 기능 로직을 유지합니다.

4.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
