# AI 동물상 테스트 사이트 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 Google의 Teachable Machine으로 생성된 인공지능 모델을 사용하여 사용자의 얼굴이 강아지상에 가까운지, 고양이상에 가까운지를 판별해주는 '동물상 테스트' 웹 애플리케이션입니다. 사용자가 자신의 사진 파일을 업로드하면, AI 모델이 얼굴을 분석하여 각 동물상에 대한 확률을 시각적으로 보여줍니다.

## 2. 주요 기능

- **AI 모델 연동:** Teachable Machine으로 학습된 강아지/고양이 이미지 분류 모델(`https://teachablemachine.withgoogle.com/models/FbPUvPxyt/`)을 사용합니다.
- **이미지 파일 업로드:** 사용자가 로컬 장치에서 이미지 파일을 선택하여 업로드할 수 있습니다.
- **결과 시각화:** 분석 결과를 각 항목(강아지, 고양이)에 대한 확률 값과 함께 진행률 표시줄(Progress Bar) 형태로 시각적으로 제공합니다.
- **테마 변경:** 사용자의 편의를 위해 다크 모드와 라이트 모드 테마 전환 기능을 제공합니다.

## 3. 개발 계획 (현재 진행 중인 변경 사항)

1.  **HTML 구조 변경 (`index.html`):**
    -   실시간 웹캠 대신 파일 업로드 방식임을 명시하도록 헤더의 설명을 수정합니다.
    -   '시작하기' 버튼을 '사진 선택' 버튼으로 변경하고, 사용자가 파일을 선택할 수 있는 숨겨진 `<input type="file">` 요소를 추가합니다.
    -   웹캠 컨테이너(`webcam-container`)를 업로드된 이미지를 표시할 `image-container`로 변경합니다.

2.  **CSS 스타일링 (`style.css`):**
    -   `#webcam-container`의 ID를 `#image-container`로 변경하고, 업로드된 이미지가 컨테이너 내에 잘 표시되도록 스타일을 조정합니다.
    -   버튼 ID 변경(`start-btn` -> `upload-btn`)을 반영합니다.

3.  **JavaScript 기능 구현 (`main.js`):**
    -   웹캠 관련 로직(`init` 함수의 웹캠 설정, `loop` 함수)을 모두 제거합니다.
    -   `init` 함수는 페이지 로드 시 AI 모델만 불러오도록 수정합니다.
    -   '사진 선택' 버튼에 클릭 이벤트를 추가하여 숨겨진 파일 입력창을 트리거합니다.
    -   파일 입력창에 `change` 이벤트를 추가하여, 사용자가 파일을 선택하면 해당 파일을 읽어 `image-container`에 미리보기로 표시하고, `predict` 함수를 호출하여 분석을 수행하도록 합니다.
    -   `predict` 함수는 이제 정적 이미지(`<img>` 요소)를 인자로 받아 분석을 수행합니다.

4.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
