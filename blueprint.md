# 제조 문의 폼 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 제조 문의를 접수하기 위한 간단한 웹 폼입니다. 사용자는 이름, 이메일, 문의 내용을 입력하여 제출할 수 있으며, Formspree 서비스를 통해 문의 내용을 수신합니다. 또한, 사용자의 편의를 위해 다크 모드와 라이트 모드 테마 전환 기능을 제공합니다.

## 2. 주요 기능

- **제조 문의 폼:** 이름, 이메일, 문의 내용을 입력할 수 있는 폼을 제공합니다.
- **Formspree 연동:** 제출된 폼 데이터는 Formspree 서비스 (`https://formspree.io/f/xzdarkow`)로 전송됩니다.
- **폼 제출 피드백:** 폼 제출 성공 또는 실패 시 사용자에게 메시지를 표시합니다.
- **테마 변경:** 사용자가 다크 모드와 라이트 모드를 선택하여 화면 테마를 변경할 수 있습니다.

## 3. 개발 계획 (현재 진행 중인 변경 사항)

1.  **HTML 구조 변경 (`index.html`):**
    -   페이지 제목을 "제조 문의 폼"으로 변경합니다.
    -   기존 로또 번호 생성기 관련 요소를 제거하고, 이름, 이메일, 문의 내용 입력 필드와 제출 버튼을 포함하는 문의 폼을 추가합니다.
    -   폼의 `action` 속성을 `https://formspree.io/f/xzdarkow`로 설정합니다.
    -   폼 제출 상태를 표시할 `p` 태그를 추가합니다.

2.  **CSS 스타일링 (`style.css`):**
    -   폼 요소 (레이블, 입력 필드, 텍스트 영역, 버튼)에 대한 스타일을 추가하고 기존의 로또 관련 스타일을 제거합니다.
    -   다크/라이트 모드 테마를 유지하고 폼 요소에도 적용합니다.

3.  **JavaScript 기능 구현 (`main.js`):**
    -   기존의 로또 번호 생성 관련 JavaScript 코드를 제거합니다.
    -   다크/라이트 모드 토글 기능은 유지합니다.
    -   폼 제출 이벤트를 가로채어 `fetch` API를 사용하여 Formspree로 데이터를 비동기적으로 전송하고, 제출 결과에 따라 사용자에게 피드백 메시지를 표시하는 기능을 구현합니다.
    -   폼 제출 성공 시 폼 필드를 초기화합니다.

4.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
