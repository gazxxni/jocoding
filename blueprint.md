# 로또 번호 추천 사이트 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 사용자에게 로또 번호를 추천해주는 웹 애플리케이션입니다. 사용자는 버튼 클릭 한 번으로 무작위 로또 번호 6개를 추천받을 수 있습니다. 또한, 사용자의 시력 보호와 편의성을 위해 다크 모드와 라이트 모드 테마 전환 기능을 제공합니다.

## 2. 주요 기능

- **로또 번호 생성:** 1부터 45까지의 숫자 중 중복되지 않는 6개의 숫자를 무작위로 생성하여 화면에 표시합니다.
- **테마 변경:** 사용자가 다크 모드와 라이트 모드를 선택하여 화면 테마를 변경할 수 있습니다.

## 3. 개발 계획

1.  **HTML 구조 변경 (`index.html`):**
    -   페이지 제목 및 헤더를 "로또 번호 생성기"로 변경합니다.
    -   번호 생성 버튼, 번호 표시 영역, 테마 변경 버튼을 추가합니다.

2.  **CSS 스타일링 (`style.css`):**
    -   라이트/다크 모드에 사용할 색상 변수를 정의합니다.
    -   전체적인 레이아웃, 버튼, 번호 표시 영역 등 각 요소에 대한 스타일을 적용합니다.
    -   `body` 요소에 클래스를 추가/제거하는 방식으로 다크 모드를 구현합니다.

3.  **JavaScript 기능 구현 (`main.js`):**
    -   로또 번호 생성 로직을 구현합니다.
    -   '번호 생성' 버튼 클릭 시 번호를 생성하고 화면에 표시하는 이벤트 리스너를 추가합니다.
    -   '테마 변경' 버튼 클릭 시 `body`의 클래스를 변경하여 테마를 전환하는 이벤트 리스너를 추가합니다.

4.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
