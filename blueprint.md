# 제조 문의 폼 및 댓글 기능 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 제조 문의를 접수하기 위한 간단한 웹 폼과 사용자 소통을 위한 댓글 기능을 제공하는 페이지입니다. 사용자는 Formspree를 통해 제조 문의를 제출할 수 있으며, Disqus를 통해 페이지 하단에서 의견을 남기고 토론에 참여할 수 있습니다. 또한, 사용자의 편의를 위해 다크 모드와 라이트 모드 테마 전환 기능을 제공합니다.

## 2. 주요 기능

- **제조 문의 폼:** 이름, 이메일, 문의 내용을 입력할 수 있는 폼을 제공합니다.
- **Formspree 연동:** 제출된 폼 데이터는 Formspree 서비스로 전송됩니다.
- **Disqus 댓글 기능:** 페이지 하단에 Disqus 댓글 섹션을 추가하여 사용자 간의 소통을 지원합니다.
- **테마 변경:** 사용자가 다크 모드와 라이트 모드를 선택하여 화면 테마를 변경할 수 있습니다.

## 3. 개발 계획 (현재 진행 중인 변경 사항)

1.  **HTML 구조 변경 (`index.html`):**
    -   제조 문의 폼 하단에 댓글 섹션을 시각적으로 구분하기 위한 `<hr>` 태그를 추가합니다.
    -   Disqus 댓글 스레드가 렌더링될 `<div id="disqus_thread"></div>` 요소를 추가합니다.

2.  **CSS 스타일링 (`style.css`):**
    -   `<hr>` 요소에 대한 스타일을 추가하여 디자인 통일성을 높입니다.

3.  **JavaScript 기능 구현 (`main.js`):**
    -   페이지 로드 시(`window.onload`) Disqus 스크립트를 동적으로 로드하고 실행하는 코드를 추가합니다.
    -   Disqus 설정 변수(`disqus_config`)를 동적으로 설정하여 현재 페이지의 URL과 고유 식별자를 제공합니다.

4.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
