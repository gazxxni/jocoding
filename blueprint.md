# AI 동물상 테스트 사이트 개발 계획

## 1. 프로젝트 개요

본 프로젝트는 Google의 Teachable Machine으로 생성된 인공지능 모델을 사용하여 사용자의 얼굴이 강아지상에 가까운지, 고양이상에 가까운지를 판별해주는 '동물상 테스트' 웹 애플리케이션입니다. shadcn/ui의 디자인 철학에 영감을 받아, 미니멀하고 모던한 UI를 통해 사용자 경험을 극대화하는 것을 목표로 합니다.

## 2. 주요 기능

- **AI 모델 연동:** Teachable Machine으로 학습된 강아지/고양이 이미지 분류 모델을 사용합니다.
- **이미지 파일 업로드:** 사용자가 로컬 장치에서 이미지 파일을 선택하여 업로드할 수 있습니다.
- **모던 UI/UX:** shadcn/ui 스타일에서 영감을 받은 깔끔하고 세련된 디자인을 적용합니다.
- **결과 시각화:** 분석 결과를 텍스트 기반의 깔끔한 형태로 제공하여 가독성을 높입니다.
- **테마 변경:** 라이트 모드와 다크 모드를 모두 지원하며, 각 모드에 최적화된 색상 팔레트를 사용합니다.

## 3. 개발 계획 (현재 진행 중인 변경 사항)

1.  **UI/UX 전면 개편 (shadcn/ui 스타일 에뮬레이션):**
    -   **색상 팔레트:** 라이트/다크 모드 각각에 대해 Zinc/Slate 테마에서 영감을 받은 새로운 색상 변수를 정의 (`--background`, `--foreground`, `--card`, `--primary` 등).
    -   **레이아웃:** 전체 페이지를 하나의 중앙 `card` 컴포넌트 구조로 변경하고, 내부에 `card-header`, `card-content`, `card-footer`로 콘텐츠를 명확히 구분합니다.
    -   **타이포그래피:** 가독성과 위계를 고려하여 폰트 크기, 굵기, 색상을 재조정합니다.
    -   **컴포넌트 스타일:**
        -   **버튼:** 기본 버튼(`button-primary`)과 고스트 버튼(`button-ghost`) 스타일을 새로 정의하여 간결함을 강조합니다.
        -   **이미지 컨테이너:** 이미지가 없을 때, 업로드 영역임을 명시하는 점선 테두리를 표시합니다.
        -   **결과 표시:** 기존의 진행률 표시줄을 제거하고, 레이블과 퍼센트 값을 `space-between`으로 정렬한 단순한 텍스트 아이템(`result-item`)으로 변경하여 가독성을 향상시킵니다.

2.  **HTML 구조 변경 (`index.html`):**
    -   새로운 CSS 구조에 맞게 `div` 구조를 `.card`, `.card-header`, `.card-content` 등으로 변경합니다.
    -   버튼에 새로운 `button` 및 `button-primary`/`button-ghost` 클래스를 적용합니다.

3.  **JavaScript 기능 구현 (`main.js`):**
    -   `predict` 함수를 수정하여, 새로운 `result-item` DOM 구조에 맞게 결과 텍스트를 생성하도록 변경합니다.

4.  **Git 버전 관리:**
    -   모든 변경 사항을 스테이징합니다.
    -   프로젝트의 변경 내용을 설명하는 커밋 메시지와 함께 커밋합니다.
